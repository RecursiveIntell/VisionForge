# VisionForge Public Claim Boundary

## Allowed now, before P31 closes

Use only prototype/work-in-progress wording:

> VisionForge is a local-first Tauri desktop prototype for image-generation workflow orchestration. It connects a React/Tauri UI with Ollama and ComfyUI-oriented pipeline, queue, gallery, and metadata/export surfaces. Current release proof is still being completed.

## Blocked until receipts exist

Do not claim:

- release-ready;
- production-ready;
- robust ComfyUI orchestration;
- verified end-to-end generation;
- tested Rust backend;
- secure/dependency-clean build;
- reproducible generation workflow;
- broad ComfyUI workflow support beyond the implemented txt2img graph;
- auto-approve/fire-and-forget unless Gate D passes.

## Upgrade path

A claim can be upgraded only after `VISIONFORGE_PROOF_PACKET.md` links it to:

- file/source evidence;
- command receipt;
- runtime smoke receipt;
- screenshot/demo evidence when user-visible;
- exact remaining limitations.
